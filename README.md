# personal-website-v2

Improvement on peronsal website by using Gatsby and React

# Test

## PreReq

Run

```
npm install
```

## Fast deployment locally

Run

```
npm run deploy
```

## Gatsby deployment locally

Run

```
gatsby build && gatsby serve
```

# Deploy

Install gh-pages

```
npm install gh-pages --save-dev
```

Run

```
npm run deploy
```
